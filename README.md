# AgroLink

React single-page app for a farm-to-region supply matching platform.

## Folder structure

```text
src/
  app/            App shell and top-level orchestration
  assets/         Local illustrations and visual assets
  components/     Shared layout and UI building blocks
  data/           Mock content and static configuration
  features/       Farmer, intermediary, and auth screens
  styles/         Theme, layout, and component styles
  utils/          Small shared helpers
```

## Run

```bash
npm install
npm run dev
```
